# test
Unit test library for C++11.
## Usage
Copy the directory `include/test` to your project.
### Include
`#include "test/test.hpp"`
### API
#### Public
namespace:`::test`  
function/class:`::test::*`  
macro:`TEST_*`  
#### Private
namespace:`::test::detail`
function/class:`::test::detail::*`  
macro:`__TEST_*`  
### Example
Refer to `tests/test_test.hpp`.
